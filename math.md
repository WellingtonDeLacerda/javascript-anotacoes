# 🧮 Math

O objeto Math fornece métodos para realizar cálculos matemáticos.

```js
Math.PI
```

Resultado:

```js
3.141592653589793
```

---

# Propriedades

## Math.PI

Retorna o valor de π (Pi).

```js
Math.PI
```

Resultado:

```js
3.141592653589793
```

---

# Métodos Básicos

## Math.max()

Retorna o maior valor.

```js
Math.max(10, 20, 30)
```

Resultado:

```js
30
```

---

## Math.min()

Retorna o menor valor.

```js
Math.min(10, 20, 30)
```

Resultado:

```js
10
```

---

## Math.abs()

Retorna o valor absoluto.

```js
Math.abs(-10)
```

Resultado:

```js
10
```

---

## Math.pow()

Eleva um número a uma potência.

```js
Math.pow(2, 3)
```

Resultado:

```js
8
```

---

## Math.sqrt()

Retorna a raiz quadrada.

```js
Math.sqrt(25)
```

Resultado:

```js
5
```

---

# Métodos de Arredondamento

## Math.round()

Arredonda para o inteiro mais próximo.

```js
Math.round(10.6)
```

Resultado:

```js
11
```

---

## Math.floor()

Arredonda para baixo.

```js
Math.floor(10.9)
```

Resultado:

```js
10
```

---

## Math.ceil()

Arredonda para cima.

```js
Math.ceil(10.1)
```

Resultado:

```js
11
```

---

## Math.trunc()

Remove a parte decimal.

```js
Math.trunc(10.99)
```

Resultado:

```js
10
```

---

# Números Aleatórios

## Math.random()

Gera um número aleatório entre 0 e 1.

```js
Math.random()
```

Resultado:

```js
0.548712...
```

---

## Número aleatório inteiro

```js
Math.floor(Math.random() * 10)
```

Resultado:

```js
0 até 9
```
